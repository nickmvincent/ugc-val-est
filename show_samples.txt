title, Search different columns in MySQL Join
body, <p>After <code>WHERE</code>, you have to specify a <em>single</em> condition, not two or three or more.</p>

<p>This condition can be constructed from simpler conditions using <a href="https://en.wikipedia.org/wiki/Logical_connective" rel="nofollow">logical operators</a> such as AND, OR, XOR, NOT.</p>

<p>For example the AND operator joins two conditions in such a way that the resulting condition is true only if both of the operands are true. So when you say</p>

</code></pre>

title, How can I declare a function that takes in a function as parameter in Scheme
body, <p>In Scheme, functions are <a href="http://en.wikipedia.org/wiki/First-class_citizen" rel="nofollow">first-class citizens</a>. Thus, you can pass a function as a parameter to another function, just like you do with any other symbol. There's no such thing as the <code>function</code> keyword.</p>

<p>For example, to define a function called <code>map</code> that takes a function as an argument, and applies it to every member of the list, you could use:</p>

<pre><code>(define (map f l)
  (if (null? l)
      l
      (cons (f (car l)) (map f (cdr l)))))
</code></pre>

<p>Then, if you had a function called <code>add1</code> that you wanted to pass to <code>map</code>:</p>

<pre><code>(define (add1 x)
  (+ x 1))

(map add1 '(1 2 3))
</code></pre>

<p>The result would be <code>(2 3 4)</code>.   </p>

<p><strong><a href="http://ideone.com/clone/V79vv" rel="nofollow">DEMO</a></strong>.</p>
