Answered posted on: 2014-06-12 09:58:02+00:00 The general class of problem is called "***secret sharing". In this instance, one solution would be to encrypt the file with a single key K, then generate and attach three encrypted versions of K, each encrypted by one combination of two DKs. (I assume that all DKs can be derived from MK.)

Note that questions like these, which are more about cryptographic protocols than about programming, would probably be better asked on crypto.
