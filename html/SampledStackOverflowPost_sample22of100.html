Most of the library cstring comparisons are set up to sort. You are looking for a difference.

So in the best K&R style, roll your own:

char *p1 = str1;
char *p2 = str2;

while (*p1 && *p2 && (*p1 == *p2))
{
    ++p1;
    ++p2;
}
// p1 and p2 should point to the two different characters (strings match if *p1 == *p2)
